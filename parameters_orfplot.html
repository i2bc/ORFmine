ORFplot parameters#

Mandatory

-tab one or several tables produced by ORFold containing the fold potential of each table entry. For each given table, ORFplot plots the distribution of the fold potential of all the table entries. ORFplot will plot as many distributions as given tables.

Optional

-h, --help shows this help message and exits

-names names of each input dataset to write in the legend. If the names consist of several words, please indicate them enclosed in double quotes, -names "Noncoding sequences (all)" "Noncoding sequences translated in condition X"
